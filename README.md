🚀 FUTURE_FS_02 — Mini CRM System

A modern full-stack Client Lead Management System built for businesses, freelancers, and agencies to manage leads, track follow-ups, and monitor conversions efficiently through a secure admin dashboard.

✨ Features
🔐 Admin Authentication
📋 Lead Management Dashboard
➕ Add / Edit / Delete Leads
📌 Lead Status Tracking
📝 Follow-up Notes
📊 Analytics Overview
🌌 Modern Animated UI
📱 Fully Responsive Design

⚙️ Installation
Clone Repository
git clone https://github.com/abitessa2711-png/FUTURE_FS_02.git
Backend Setup
cd backend
npm install
npm start

Create .env

PORT=5000
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=yourpassword
MYSQL_DATABASE=mini_crm
JWT_SECRET=your_secret_key
Frontend Setup
cd frontend
npm install
npm run dev
🗄️ Database Setup

Import:

backend/setup.sql
🌍 Deployment
Frontend → Vercel / Netlify
Backend → Render / Railway
